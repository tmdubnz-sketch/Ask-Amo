# 📚 Ask-Amo Comprehensive User Guide

## 🎯 **Overview**

Ask-Amo is a sophisticated AI assistant with 7 specialized views designed for different tasks. Amo can chat, browse the web, run terminal commands, edit code, build vocabulary, create sentences, and enhance intent understanding.

## 🗺️ **Navigation & Views**

### **Main Navigation**
- **7 Top Tabs**: Chat, Web Browser, Terminal, Code Editor, Vocabulary, Sentence Builder, Intent Enhancer
- **Horizontal Scrolling**: All tabs accessible with smooth scrolling on any screen size
- **Voice Commands**: Navigate hands-free with voice commands
- **Quick Access**: Sidebar shortcuts for rapid switching

### **View Switching Commands**
```
go to chat                    → Main conversation interface
go to webview / open browser   → Web browsing interface  
go to terminal                 → Shell command execution
go to editor                   → Code editing interface
go to vocabulary               → Vocabulary building tools
go to sentence builder         → Sentence creation tools
go to intent enhancer          → Intent prediction tools
```
