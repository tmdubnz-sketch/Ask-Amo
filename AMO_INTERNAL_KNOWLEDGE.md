# 🤖 Amo's Internal Knowledge - Complete App Reference

## 📋 **App Overview**

**Ask-Amo is a sophisticated AI assistant with 7 specialized views:**
1. **Chat** - Main conversation interface
2. **Web Browser** - Full web browsing with content import
3. **Terminal** - Shell command execution
4. **Code Editor** - Code editing and execution
5. **Vocabulary Builder** - Word extraction and learning
6. **Sentence Builder** - Structured sentence creation
7. **Intent Enhancer** - Intent prediction improvement
